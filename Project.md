# アルゼンチンタンゴ・イントロクイズ 開発プロジェクト

---

## 1. プロジェクト概要

- **アプリ名**: タンゴ・クイズ (tango-quiz)
- **目的**: ミロンガで流れる曲名が覚えられない悩みを解決するため、イントロを聴いて「曲名・アーティスト・年代」を当てるクイズ形式の学習アプリ
- **ターゲット**: タンゴ愛好家、ダンサー（特に曲名を覚えたい人）
- **GitHubリポジトリ**: taaku-tech/tango-quiz
- **ホスティング**: Vercel

---

## 2. 開発環境

- エディタ: VS Code
- AI支援: Claude Code
- 実行環境: WSL/Ubuntu
- ソース管理: GitHub (taaku-tech/tango-quiz)

---

## 3. 実装済み機能

### クイズ機能
- **通常クイズ**: 全58曲（15楽団）からランダムに10問出題する4択クイズ
- **演奏家別クイズ**: 特定演奏家の曲のみを対象とした集中クイズ（4曲以上収録の演奏家のみ対応）

### コンテンツ
- **演奏家紹介ページ**: 15名の演奏家の特徴・ダンスのポイント・代表曲を掲載
- **楽団系譜図ページ**: 1897〜1998年の楽団の師弟・親交関係をインタラクティブSVGで可視化

### UX機能
- **TTS（音声読み上げ）**: 選択肢の曲名・演奏家名・年代をブラウザのSpeech APIで読み上げ
- **Spotify埋め込みプレイヤー**: blur処理でイントロ当てに集中できる仕組み
- **曲解説モーダル**: 回答後および結果画面から曲の背景解説を閲覧可能
